# Desktop AI Agents

A curated list of **on-device AI agent runners and harnesses** — tools that run the agent loop locally on your desktop, laptop, or workstation rather than in the cloud.

The unifying theme: **the agent runs on your device.** The underlying model may be a hosted API (Anthropic, OpenAI, xAI, etc.) or a fully local model — what matters is that the harness, tool execution, and control loop live on your machine.
