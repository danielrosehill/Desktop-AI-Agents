# Desktop-AI-Agents

Curated resource list of **on-device AI agent runners and harnesses** — tools like opcode, grok-cli, Claude Code, aider, etc. that run agents locally on the user's machine rather than in the cloud.

## Scope

- Agent harnesses / runners that execute on the user's desktop, laptop, or workstation
- CLIs, TUIs, and GUI desktop apps all qualify
- The unifying theme: **the agent loop runs on-device**, even if the underlying model is a hosted API
- Fully local (model + agent) tools are in scope and worth flagging

Out of scope: cloud-only agent platforms, hosted SaaS agents, browser-only playgrounds.

## Formatting preferences

- Organize by category (e.g., CLI Harnesses, Desktop GUI Agents, Local-Model Runners, IDE-Integrated Agents)
- Projects sorted alphabetically within each section
- Each entry uses shields.io badges (stars, last commit)
- Keep descriptions short (1–2 sentences)
- Include language and author link
- No endorsement implied; non-exhaustive best-effort list
